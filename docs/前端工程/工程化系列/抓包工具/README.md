# 抓包工具

## charles

* 支持SSL代理，可以截取分析SSL的请求
* 支持流量控制。可以模拟慢速网络(2G,3G)，以及等待时间较长的请求。
* 支持AJAX调试。可以自动把JSON或者XML数据格式化，方便查看。
* 支持重发网络请求，方便后端调试。
* 支持修改网络请求参数。
* 支持网络请求的截取和动态修改。
* 最重要的一个优点就是有不同平台的版本（Mac，Windows、Linux）即学一个打遍天下。
