# 镜像：

镜像可以理解为互联网上的软件包的下载地址

查看当前使用的镜像源：

yarn config get registry

切换当前镜像：

npm  install  -g nrm

查看所有镜像源：

nrm ls

用指定镜像源：

nrm use taobao

测试镜像速度：

nrm test taobao
