# 子组件方法只允许执行一次

# 应用场景

在初始化子组件时需要为某个元素绑定一个监听元素滚动条滚动的滚动事件。子组件只有在显示的时候才会获取相应的数据渲染。

如何实现让子组件显示出来并且 dom 加载完成之后去再去绑定滚动事件，且只执行一次。
